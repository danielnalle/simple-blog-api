# Simple Blog API

## Overview

Simple Blog API. Literally like the name, you can create, read, update, and delete a post. It has login and register for authentication. You can add a comment and delete a comment from a post. This is my first (serious âœ¨) project to become a Backend Developer.

## Tech Stack

- Javascript (Node.JS) as Programming Language
- Express JS to build API
- PostgreSQL for database
- JWT for authentication
- Bcrypt for hash password

## How To Run

1. Clone this repo and install all dependency

   ```
   github clone https://github.com/danielnalle/simple-blog-api
   cd simple-blog-api
   npm i
   ```

2. Rename `.env.example` to `.env` and modify all variables based on your environment

3. Run locally

   ```
    npm run dev
   ```

4. You can check all API documentation in [here](docs)
